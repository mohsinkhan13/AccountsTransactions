You have to do the following tasks:

  1. add the following line to global.asax.cs\Application_Start
    GlobalConfiguration.Configuration.RegisterWebSecurity();
  2. add the following references to web.config
    system.identityModel: configSource="Configuration\identitymodel.config" system.identityModel.services configSource="Configuration\identitymodel.services.config"
  3. add the following transformations to web.config.addison/web.config.portalbereich
    Configuration\...\identitymodel.config Configuration\...\identitymodel.services.config
